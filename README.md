# Pascoal Distribuidora
Pascoal Distribuidora é uma aplicação web moderna de e-commerce construída com Vite, React (JavaScript) e Tailwind CSS.  O projeto simula uma plataforma de venda e entrega de alimentos congelados, como peixe, frango, carne, bebidas e outros produtos alimentares.  Funcionalidades: - Interface moderna e responsiva (mobile first) - Catálogo de produtos com cards dinâmicos - Sistema de carrinho de compras - Atualização automática de quantidades e total - Finalização de pedidos via WhatsApp - Estrutura preparada para integração com backend futuramente  Tecnologias: - Vite - React (JavaScript) - Tailwind CSS  Objetivo: Criar uma base sólida para um sistema real de distribuição de alimentos, focado em vendas rápidas, experiência simples e escalabilidade futura.
